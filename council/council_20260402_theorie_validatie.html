Council: Theorie Validatie

2 april 2026 — 5 adviseurs + 5 peer reviewers

Vraag: We hebben Cannae's strategie reverse-engineered: een bot die 5-6 dagen voor wedstrijden Win NO + Draw YES koopt op PM voetbal. ROI ~21.7% op $25.2M. Sizing = conviction-signaal. Valideer de theorie en identificeer wat er mist voordat we paper trades starten.

De 5 Adviseurs
🧠
Eerste Principes Denker
Data vervuild, observatie ≠ verklaring, verliezers ontbreken

1. Data is vervuild. Survivorship bias maakt 21.7% ROI een bovengrens. Hauptbet-analyse (64.5-68.4% WR) is realistischer, maar ROI niet herberekend op dat niveau.

2. Win NO + Draw YES = observatie. WAT Cannae doet is beschreven, niet WAAROM. Informatievoorsprong of structureel marktdefect?

3. Verliezers ontbreken. Q1-Q4 bets met 57% WR en 4-8% ROI = nauwelijks boven break-even. We kopieren ook de marginale bets.

Gaps: hauptbet P&L, verliesanalyse, timing-ROI relatie, slippage.

🚀
Expansionist
Counterfactual baseline, calibration curve, league-timing

1. Counter-factual ontbreekt. Naieve Win NO + Draw YES simulatie = baseline. Verschil met Cannae = zijn echte alpha.

2. Calibration curve ontbreekt. avgPrice vs settlement per league. Koopt hij underpriced? 0.55 avg = 45ct edge. 0.85 avg = dunne edge.

3. League-specifieke timing. 5.7 dagen is gemiddeld. EPL vs CL kan totaal anders. Split per league.

Kost 0 API credits — alleen bestaande dataset nodig.

👀
Buitenstaander
21.7% op $25M te mooi, $3K ≠ $3.56M, onzichtbare verliezen

21.7% op $25.2M klinkt te mooi. Zonder tijdframe zegt het niets. Fee-structuur en slippage bij die volumes?

Survivorship correctie halfbakken. Correctie naar 65% is een gok. 65% op 50c tokens = breakeven na fees.

Sizing ≠ skill. "Meer inzet = hogere winkans" = basale bankroll management. De edge moet voorspelbaar zijn.

$3K vs $3.56M = ander spel. Paper trading valideert weinig, risico zit in executie.

⚠
Contrariaan
Conviction-sizing circulair, Win NO ROI opgeblazen, edge dalend?

Circulaire redenering. Grote bets op goedkope tokens (30-50c) = inherent hogere ROI. +0.107 correlatie = <2% variantie. Je meet wiskunde, niet skill.

Win NO ROI opgeblazen. 40.6% op $6.6M impliceert PM overschat systematisch team-winkansen. Retail favoriet-bias? Edge verdampt als meer bots het ontdekken.

Gaps: ROI tijdserie (dalend?), Draw YES sample te klein ($0.8M), Pinnacle op 12 posities = anekdotisch, taker fee 0.44% vs maker 0%.

🔨
Uitvoerder
Stop backward-looking. Live mispricing meten. Forward-test.

Stap 1: Live mispricing (2-3 uur) — PM vs Odds API voor alle EPL/BuLi/CL. Geen mispricing = stop.

Stap 2: Forward-test (1-2 dagen) — log-only, welke wedstrijden mispricing tonen.

Stap 3: Paper trade (1 dag) — drempel bepalen, /opt/bottie-test/ configureren.

NIET doen: historische odds, post-entry analyse. Die data heb je niet. Forward = enige optie.

Verdict: Solide hypothese, geen bewezen strategie
De claim "Cannae exploiteert PM's favoriet-bias via Win NO + Draw YES" is consistent met de data maar niet gefalsifieerd. Drie risico's voor onze bot:

Kopieerbaar Edge is structureel (2/3 uitkomsten, PM onderprijst) → counterfactual bewijst dit

Niet kopieerbaar Edge zit in timing/fees (maker-side, spread-capture) → taker betaalt 0.44% meer

Onzichtbaar Edge zit in selectiviteit (Cannae slaat wedstrijden over) → niet in onze data

Blinde Vlekken (Peer Reviews)
1. Fee-asymmetrie

Cannae = maker (0% + rebate). Wij = taker (0.44%). Netto-ROI 1-2% lager. Bij realistische 5-8% hauptbet ROI = dodelijk.

2. Market Impact

$25.2M volume beweegt de markt. Copybot koopt tegen door-Cannae-bewogen prijzen. Edge deels verdampt bij instap.

3. Selectiviteits-bias

We analyseren Cannae OMDAT hij succesvol is. Hoeveel wallets met dezelfde strategie faalden? Base rate onbekend.

4. Liquiditeits-timing

Vroeg kopen bij brede spread = maker = negatieve fee via rebate. Verklaart hoge ROI zonder informatieve edge.

Eerste Stappen (PARALLEL, vandaag)

A. Counterfactual baseline (2 uur, 0 credits)

Naief Win NO + Draw YES op alle EPL/BuLi/CL, flat sizing, avgPrice entry. Baseline >15% = structureel. <5% = Cannae's alpha zit elders.

B. Live mispricing scan (1 uur, ~10 credits)

PM prijs vs bookmaker implied prob voor komende wedstrijden. Mispricing >5% = theorie leeft. <2% = markt efficient.

C. Verliesanalyse (1 uur, 0 credits)

Kenmerken verliezende hauptbets: league, timing, sizing, avgPrice. Patronen zoeken om te filteren.

Na deze drie analyses: go/no-go beslissing voor paper trading.
