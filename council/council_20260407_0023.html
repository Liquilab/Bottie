Peer Reviews — Wat de adviseurs misten

Drie van de vijf reviewers vingen onafhankelijk dezelfde miss op: niemand heeft CLV (closing price) gemeten op Cannae's eigen legs. Bottie heeft die tracking al sinds commit 145e8c0. Zonder CLV is +5.2% ROI mogelijk mid-price drift, geen edge.

Andere blinde vlekken: look-ahead bias (config opgebouwd uit dezelfde 42d → zero out-of-sample informatiewaarde) · effectieve n is 368, niet 561 (1.5 legs/game = niet onafhankelijk) · geen Cannae source-monitoring (regime-shift detection ontbreekt) · slippage op NO-favorieten kan 1-2% per tick kosten · mogelijke overlap met al-live blind_fade.

Sterkste antwoorden: 3 reviewers kozen D (Eerste Principes — subtractief), 1 koos B (Contrariaan), 1 koos C (Buitenstaander). Twee reviewers wezen A (Uitvoerder) aan als grootste blinde vlek (te snel live zonder validatie). Eén wees E (Expansionist) aan (vloot van drie schaalt noise).

Verdict

Doe de subtractieve fix (D) met de concrete config van A, MAAR voeg twee guardrails toe vóór deploy.
  1. Strict subtractief, niet expansief. Blacklist epl, lal, mlb. Skip WIN_YES solo (alleen kopiëren met DRAW≥$2k). Houd 3% flat sizing. Geen sizing bumps — geen "WIN_NO is +17.9% dus 6%". Expansionist wordt afgewezen: vloot van drie bots op 42d data is variantie schalen, niet edge.
  2. CLV-validatie als pre-deploy gate. Draai retro op bestaande closing_price data in data/trades.jsonl voor de subset (whitelist minus EPL/LAL/MLB, WIN_NO + WIN_YES-met-draw). Als Cannae's CLV op die subset positief is op n>50 → deploy. Anders: het +17.9% getal is achterafperspectief.
  3. Source-monitoring kill-switch. daily_loss_limit: -500 (strenger dan A's -1000), max_daily_trades: 10, pause_on_consecutive_losses: 4. Strenger omdat statistische ondergrond zwakker is dan A erkent.
Eerste stap (vandaag): Run CLV-validatie op de subset (data/trades.jsonl, filter whitelist minus epl/lal/mlb, splits WIN_NO solo / WIN_YES+DRAW / rest, bereken Σ(closing − entry) × shares per groep). Als WIN_NO solo CLV-positief op n>50 → groen licht voor deploy met bovenstaande config morgenochtend.
